## devbox deploy config

Copy template configs to service directory

### Synopsis

Copy config file templates declared in services.<service>.configs from
configs/services/<service>/ to services/<service>/configs/.

Mode controls copy behavior: 'default' skips existing files, 'update' copies always, 'replace' overwrites.

```
devbox deploy config <service> [flags]
```

### Examples

```
  devbox deploy config main
  devbox deploy config main --mode update
```
