## devbox stop

Stop compose services without removing them

### Synopsis

Stop running compose services without removing their containers.

When service names are provided, only those services are stopped.
Use 'devbox down' to stop and remove containers.

```
devbox stop [services...] [flags]
```

  devbox stop app-main
