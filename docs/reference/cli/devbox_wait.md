## devbox wait

Wait for all compose containers to become healthy

### Synopsis

Poll all running compose containers until they report a healthy status or the timeout elapses.

Containers without a healthcheck are considered healthy immediately.
Use '--timeout' and '--interval' to control polling behavior.

```
devbox wait [flags]
```

### Examples

```
  devbox wait
  devbox wait --timeout 120s --interval 5s
```
