## devbox run

Start the project (full lifecycle: update probe → up → wait → info)

### Synopsis

Run the full project lifecycle driven by devbox/lifecycle.yml.

Execution order: optional git update probe → before-run hooks → docker up → docker wait
→ after-run hooks → optional info display → final ready message.

Use 'devbox up' for a bare Docker Compose start without hooks or the update probe.
Use 'devbox docker up' for the low-level compose control plane.

```
devbox run [flags]
```

### Examples

```
  devbox run
  devbox run --no-update
  devbox run --update auto
```

### Options

```
  -h, --help            help for run
      --no-update       disable git update probe regardless of lifecycle.yml config
      --update string   override update probe mode (prompt|auto|check|off)
  -y, --yes             skip confirmation prompts inside hook steps
```
